## Installing

Add this package to your flutter project's `pubspec.yaml` file:

```yaml
dependencies:
    ...
    advanced_icon: <latest version>
```

Now in your `Dart` code, you can use:

```dart
import 'package:advanced_icon/advanced_icon.dart';
```
